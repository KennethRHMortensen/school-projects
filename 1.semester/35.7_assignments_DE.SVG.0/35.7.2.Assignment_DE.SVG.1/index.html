What is a viewbox?

The viewBox is similar to the viewport, but you can also use it to “pan” and “zoom” like a telescope.
viewBox got 4 attributes, with the first two being control of its "panning", and the last two being control of its "zoom":

  • min-x
  • min-y
  • width
  • height

What is a viewport?

The viewport is like a window you look through to see an SVG’s content, basically it creates a “port” through which you can “view” a section of an SVG.
you can control the viewport via width and height parameters on the svg element.
For example, you might have a shape in your graphic that is 100px by 100px, but if you set the SVG viewport to 50px by 50px you’ll only see a portion of that shape.

How do they relate to each other?

The viewBox is similar to the viewport, but you can also use it to “pan” and “zoom” like a telescope.
This means that if the value of width and height of the viewport is less than the last two attribute values of the viewBox, it will zoom out and make the object drawn smaller within the viewBox.
Same goes the opposite way if viewBox last two values are smaller than the viewports width and height, it will zoom in on the drawn object, meaning you might won't see its full size.

How to scale an SVG drawing's appearance on a web page?

By adding a viewBox to its attributes, this way it will scale perfectly to fit within what ever size you give the container that contains the svg.
By default it will not be stretched or distorted if you give it dimensions that doesn't match its aspect ratio. Instead, the scale will be adjusted in order to preserve the aspect ratio defined in the code.
